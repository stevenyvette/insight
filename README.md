# Insight——A Network Visualization Platform 
---
#使用说明
- 本平台为在线可视化平台
- 由于安全性限制，组件需要***服务器环境***才可运行
- 使用方法：
	- 搭建服务器
	- 将本文件夹作为根目录放到服务期目录下
	- 浏览相应网址进入
	- 目前在线地址： [Insight](http://120.25.166.203/insight/)
